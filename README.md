# 1c-merge-report

Локальное оффлайн веб-приложение для разбора TXT-отчёта сравнения конфигураций 1С и согласования мерджа.

## Как открыть

1. Скачайте/склонируйте проект.
2. Откройте `index.html` в браузере (интернет не нужен).
3. Нажмите **«Загрузить TXT»** и выберите отчёт.

## Как пользоваться

- Слева отображается дерево: `Раздел → Тип метаданных → Объект`.
- Справа карточка объекта:
  - Таблица метаданных: контекст/свойство/было/стало.
  - Блоки кода (hunks): было/стало, модуль, процедура/функция.
  - Комментарии к объекту и каждому отличию.
- Вкладки:
  - **Основные изменения**
  - **Справочная информация (некритично)**
- Фильтры: поиск, раздел, уровень риска.
- Экспорт:
  - **Сохранить сессию** — JSON с комментариями.
  - **Загрузить сессию** — восстановление JSON.
  - **CSV: Основные** и **CSV: Справка**.

## Кэш

Приложение автоматически сохраняет текущую сессию (включая комментарии) в `localStorage`.

## Входной формат

Парсер ищет объекты по метаданному пути (`Документ.`, `Справочник.`, `ОбщийМодуль.` и др.) и маркерам `***`, `-->`, `<---`.
Поддерживается группировка изменений в hunks на основе `<`/`>` и контекста (модули, процедуры/функции, реквизиты и т.д.).
